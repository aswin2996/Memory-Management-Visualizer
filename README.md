# Dynamic Memory Management Visualizer

This is a simple full-stack web application designed to simulate concepts learned in an Operating Systems course (Paging, Segmentation, and Virtual Memory).

## Features

1. **Paging Simulation**: Visualize FIFO and LRU page replacement algorithms step-by-step.
2. **Segmentation**: Calculate physical addresses using a simple segment table.
3. **Virtual Memory**: Basic simulation of limited physical frames and swap space.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Python, Flask

## How to Run
Just double-click `start.bat` on Windows! It will automatically install the necessary Python packages and start the Flask server.

Alternatively:
```bash
pip install -r requirements.txt
python app.py
```

Then open `http://127.0.0.1:5000/` in your web browser.
